1. See which ports are running on linux: `sudo lsof -i -P -n | grep LISTEN`
2. Undo uncommitted changes (revert to current commit): `git reset --hard HEAD`
3. Install Docker on AWS:
    - `sudo yum update -y`
    - `sudo amazon-linux-extras install docker`
	- `sudo service docker start`
    - `sudo usermod -a -G docker ec2-user`
